Advent of Code 2025 in Charly

1. December 2025

This year I’m semi-comitting to implementing each Advent of Code challenge in my programming language Charly.

If you take a look at the Charly standard library, you will notice the following:

It is barren

The stdlib lacks the vast majority of features that a modern language would normally provide. Adding all the necessary building blocks to solve the daily puzzles thus becomes part of the challenge.

Because of the compounding effect of this, there might be days where no changes are required, and days where I’ll have to reinvent the wheel to get things running. I might not post every day, only on days where something interesting has been built.

Each article contains a short summary of how I solved the puzzle and what changes I had to make to the stdlib or the VM itself.
